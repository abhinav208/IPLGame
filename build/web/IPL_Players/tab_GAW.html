Guess And Win

  • Each match will have its own predictors challenge questions
    ex. Man of the match, max wicket taker, max sixes ...
  • You can predict these questions.
  • There is no negative rating
  • Results of these will be declared along with bid results.
  • 100 points for each correct answer.
  • If no one gives correct Answer, then nearest answer will also be considered

    Ex: If Correct Answer is 5 and none of the user predicted 5
    And nearest provided answer is 4 and 6, then both will be considered correct

  • If case of more than one answer, then and condition will be applied

    Ex: Which Bowler will make a clean bowled ?
    Answer provided is "Sanju Samson, Mohit Sharma"
    if both makes clean bowled then answer will be considered Correct
    Any one makes clean bowled then answer will not be considered Correct
